Développement Mobile

Description :

Le développement mobile forme les étudiants à la création d'applications pour smartphones et tablettes, principalement sous Android.

Points Clés :

  • Langages : Kotlin, Java (Android) ou Swift (iOS).
  • Outils : Android Studio.
  • Accès aux fonctionnalités du téléphone : caméra, GPS, stockage.

Projets :

Création d'applications fonctionnelles (ex. : gestion des tâches, suivi de la santé).
