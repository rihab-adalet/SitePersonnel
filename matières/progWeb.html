Programmation Web

Description :

La programmation web forme les étudiants à la création de sites web et d'applications interactives. Ce cours se concentre sur les technologies front-end (interface utilisateur) et back-end (serveur).

Points Clés :

  • Front-End : HTML5, CSS3, JavaScript, Bootstrap.
  • Back-End : PHP, Node.js, ou frameworks comme Django/Flask.
  • Connexion à une base de données.

Projets :

Création de sites dynamiques, formulaires interactifs, et applications web.
